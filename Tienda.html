Tienda

La tienda son los comandos que se pueden realizar para comprar huevos o lucky's, cada huevo puede comprarse con su respectivo nivel de rareza, mientras que los luckys actuan por mineral, el precio es determinado entre mayor sea el nivel de rareza y la moneda usada son los puntos, para visualizar tus puntos esta el comando !puntos y para visualizar estos mismos precios puedes usar el comando !tienda, para comprar puedes usar el comando !comprar huevo_[rareza] o en caso de ser un lucky !comprar [material]

Puedes usar el comando !craftear [rareza] para los huevos en lugar de comprarlos, para revisar tus ingredientes usa el comando !tengo, tambien puedes craftear todo lo disponible usando el comando !craftear todo
