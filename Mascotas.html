Mascotas

Las mascotas las podemos encontrar abriendo huevos dependiendo de su calidad mediante, puedes mirarlo con !mascota [Id], se puede ganar experiencia mediante la pesca para luego darsela a una mascota mediante el comando !entrenar [Id] [Cantidad de xp], cada mascota requiere una cantidad diferente de xp para subir de nivel segun su rareza y su nivel actual, estas mejoras tambien pueden ser vistas mediante el comando !mejoras [Id].

Puedes revisar tu xp acomulada mediante el comando !xp

Cada mascota requiere un slot para equiparla, mediante el comando !comprar_slot hasta un maximo de 12, luego de comprarlo necesitas equipar la mascota mediante el comando !equipar [Id] [Numero de Slot]
